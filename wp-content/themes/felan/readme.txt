Install Steps
--------------
Step 1: Go to Appearance > Add New Themes, then click Upload button

Step 2: Go to browse, then select the felan.zip file

Step 3: Click "Install Now" to install Felan

Step 4: Activate the newly installed Felan by going to Appearance > Themes, find Felan and click on "Activate" button

Step 5: Once Felan is activated, you will be redirected to Felan Welcome Screen. Switch to Plugins tab to install plugins.

Step 6: click "Install" to begin installing plugins. After that, you will have to activate plugins.
